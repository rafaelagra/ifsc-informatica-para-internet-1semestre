# ifsc-informatica-para-internet-1semestre
Repositório com as atividades mão na massa práticas desenvolvidas no 1º semestre do curso Técnico em Informática para Internet do IFSC. Contém atividades em lógica de programação, backend (Java), frontend (HTML, CSS e JavaScript), banco de dados (MySQL) e análise de sistemas (UML), consolidando fundamentos de desenvolvimento web.
