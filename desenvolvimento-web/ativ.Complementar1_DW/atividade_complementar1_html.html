Primeira atividade complementar de desenvolvimento web I

Atividade prática complementar de html5

Requisitos para a realização da atividade:

O documento web deve representar uma página completa, com doctype, os marcadores básicos da seção head e elementos solicitados abaixo, na seção body.

O documento web deve apresentar um título, a ser exibido na aba do navegador.

O documento web deve apresentar um texto atuando como cabeçalho da página web, representando a informação mais importante a ser percebida pelo usuário da página web.

O documento web deve apresentar a marcação de, no mínimo, dois parágrafos de texto regular. Use o texto que desejar.
